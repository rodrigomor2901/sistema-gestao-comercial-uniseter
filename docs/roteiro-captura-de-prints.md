# Roteiro de Captura de Prints

## Objetivo

Este roteiro organiza a próxima rodada de capturas para concluir a versão final ilustrada do material de treinamento.

---

## 1. Prints já capturados

- Dashboard
- Funil de vendas
- Solicitações
- Bloco de postos e equipamentos por serviço
- Fila de propostas
- Geração do número da proposta
- Triagem com mover para etapa e PDF
- Negociações
- Negociações com probabilidade e resumo do dia
- Contratos
- Contratos com cláusulas e assinatura
- Administrar usuários
- Configurações de listas

---

## 2. Prints ainda recomendados

### Administração

- formulário de criação/edição de usuário
- exemplo de manutenção de lista configurável

### Propostas

- tela com proposta finalizada e anexo PDF em caso real concluído

### Negociações

- tela de desfecho comercial com aceite/anexo, se quiser complementar

### Contratos

- tela com caso real preenchido em elaboração de contrato
- tela com caso real preenchido em negociação de cláusulas
- tela com caso real preenchido em contrato assinado com anexo

---

## 3. Padrão sugerido para cada print

Para cada módulo, preferir:

1. print geral da tela
2. print aproximado dos campos principais
3. print do botão ou ação principal
4. print do status final esperado
